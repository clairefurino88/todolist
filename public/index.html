Your To Do List!

We all have those days when there are a million things to do, and we don't know how we're going to get it all done. It's easy to become overwhelmed by the vast quantity of tasks that we must do from day to day. This is why the To Do List App can be so helpful. When we get too busy, we end up feeling like we are barely able to keep our heads above water. This applicaiton allows the users to create a list of things to do. The check mark can be clicked when a task is completed and a line will be drawn through the text. The X can be clicked to delete a task, and if you hover over the task, you can click on it to edit/change the task at hand. All of these tasks will be stored in a database
